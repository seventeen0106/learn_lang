# 一、Agent 基礎

## 1.1 複合式 AI 系統

諸如：

- 檢索增強系統（RAG）
    
- 對話式 AI（Conversation AI）
    
- 人機互動（Copilts）
    

## 1.2 關鍵要素

一個完整的 Agent通常具備四大關鍵能力

1. **思考分析能力：**
    
    最核心且首要的能力。能將複雜任務分解成小步驟，並制定出完整的執行計劃。
    
2. **規劃能力：**
    
    形成子目標並建構從當前狀態到未來狀態的路徑的決策技能。
    
3. **工具(Tool)運用能力：**
    
    指的是 Agent 本身之外的程式或功能。Agent 需要判斷何時及如何調用這些外部工具。這些工具可能是資料庫查詢、計算函數，甚至是其他語言模型。
    
4. **資訊保存能力（記憶）：**
    
    (1) Agent 在思考過程中產生的內部日誌，在需要時調用。 (2)Agent與使用者之間的對話歷史，更加個性化和連貫的互動。
    

## 1.3 Agent v.s. Agentic

### Agent

這個名詞通常指的是一個具有特定功能的獨立實體。在AI領域中，它可能是一個執行預定任務的程序或系統

### Agentic

是一個形容詞，描述的是系統表現出的類似 agent 的特性程度。這種區別不是簡單的二元對立，而是一個連續的光譜

## 1.4 Agent v.s. Chatbot

1. 能夠獨立評估情境，制定計劃並執行行動，以達成預期目標，不依賴固定腳本
    
2. 能夠理解對話的更廣泛情境，包括使用者的情緒、偏好和過去的互動，提供更個性化且相關性更高的回應。
    
3. 從每次互動中學習，不斷優化
    
4. 能夠無縫整合並處理多種輸入和輸出模態，如文字、語音和視覺數據
    

# 二、模式

## 2.1 最知名：ReAct

### 反應(Reaction)：

Agent 收到任務後，先對任務進行一番審視。

### 行爲(Action)：

開始對任務逐一展開動作。這裡可以直接回答會是調用工具。  
結束後觀察結果是否完成目標，不夠再回到步驟 A。
